Element

Elements are like keywords that tells the browser how to format text and apperence on the website.

Example of those tags could be, H1, Header, Footer, Button which is parts of the functions of a website.

HTML

HTML stands for Hyper markup Language and is the standard markuplanguage for todays websites.

Using the language you can stucture up the website and make it look the way you want it to look.

ID

The ID module is used to give elements a specific identification. When giving a element an ID you can

use your CSS to customize its appearance. For example changing the color, size, and font of the text.

Class

Class is similar to ID but the main difference between them is that ID is more specific for an element.

Class can be used multiple times on multiple parts of your index.html file. Class can also be used to change appearance like ID.

Meta

The meta is used to define the metadata for a website and it is often located in the “head” tag at the top of an HTML file.

The meta section is often used to specify a character set, viewport which is used to make a website responsive.

Responsive

When making a website your goal is to make it as responsive as possible, which means you want your website

to “fit” all types of devices and all aspect ratios. This is easiest achieved using Bootstrap and CSS

http

Stands for Hypertext Transfer Protocol and is a protocol that is used on a bunch of websites out there.

It allows users to communicate data on the WWW - ” World Wide Web”

https

Stands for Hypertext Transfer Protocol Secure and is a protocol similar to HTTP. This protocol is used

to transfer data on a secure route. Websites having HTTPS are secure and there’s a minimal risk of privacy getting

leaked, and sensitive personal information getting out.

compressed data

Data compression is the process of modifying, encoding, or converting the structure of the bits of data in a way that consumes

less space on the disk. It makes it possible to reduce the storage needed to save the data or the elements of a website.

CDN

Stands for Content delivery network and is often referred to as a couple of servers helping each

other to minimize delays in loading content on a webpage. This is done by having the servers closer to you.

local bootstrap

Bootstrap is a library template for basic css. Instead of having to type every element, piece by piece, it is a downloadable

CSS file already done. Local bootstrap is exacly the same, bootstrap can be runned in a local state, on the computer or an

online state, where you link web to web.

JQuery

JQuery is a library similar to local bootstrap but for JavaScript functions. It is used as a template and can easily be

costumized by the coder. JavaScript can be used to modify parts of your website and is used to createthe button I have

in the right bottom corner.

Flexbox

Flexbox is a layout model and allow your website to align, scale and respond in a specific way choosen by the coder.

You can costumize things like height, width and space between your texts using this layout model.

Code snippets

It is a small snippet of code reusable for another part of the website, an example could be a navbar, which is a returning

part of a website. The navbar can be reused and you could only in need to change small parts of it, like the text and so on.

Grid

Grid has the same concept as flexbox, but instead of being as visual as flexbox it is used behind the scenes

to devide a page to different parts and regions. With grid you devide elements by using the class tag.
